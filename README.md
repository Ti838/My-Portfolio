## 🚀 Features

- **Project Showcase:** Dynamic grid showing my latest GitHub activity and featured contributions.
- **Secure Auth:** Integrated authentication system for private sections and admin controls.
- **Responsive Design:** Optimized for a seamless experience across mobile, tablet, and desktop.
- **Blog Section:** Integrated markdown-based blog for sharing technical insights and tutorials.

## 💻 Powered By

- **Framework:** React / TypeScript
- **Styling:** Vanilla CSS (Custom UI Components)
- **Deployment:** Vercel

## ⚙️ Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
